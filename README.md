A complete ride-hailing web application (similar to Uber/PickMe) with three interfaces: Passenger, Driver, and Admin.

## Overview

- **Passenger App** — book rides, track drivers live, view ride history
- **Driver App** — accept rides, navigate, track earnings
- **Admin Dashboard** — manage the platform, monitor rides, configure pricing

See [`src/PROJECT_DOCUMENTATION.md`](src/PROJECT_DOCUMENTATION.md) for the full feature list and project structure.

## Running it

```bash
npm i
npm run dev
```
